# pingpong
игра пингпонг
В репозитории лежит игра пинг-понг.
Игра является мультиплеерной(2 человека по локальной сети)
Реализована на языке Python
Каждый из игроков управляет ракеткой, которая отталкивает мяч
Главной целью является забить шар в ворота, также есть счетчик, по истечении 3 минут  побеждает человек с большим количеством очков.



ping pong game
The repository contains a ping pong game.
The game is multiplayer (2 people on a local network)
Implemented in Python
Each player controls a paddle that pushes the ball
The main goal is to score a ball into the goal, and there is a counter that ends after 3 minutes, and the person with the most points wins.






乒乓球游戏
存储库中有一个乒乓球游戏。
游戏是多人游戏（本地网络上的2人）
用Python实现。
每个球员都控制一个球拍，将球推开
主要目标是在球门内得分，还有一个计数器，3分钟后得分最多的人获胜。

Python тілінде енгізілген

spiel pingpong
Im Repository liegt ein Ping-Pong-Spiel.
Das Spiel ist Multiplayer(2 Personen über ein lokales Netzwerk)
Implementiert in Python
Jeder Spieler steuert einen Schläger, der den Ball abstößt
Das Hauptziel ist es, den Ball ins Tor zu schießen, es gibt auch einen Zähler, nach 3 Minuten gewinnt eine Person mit vielen Punkten.
